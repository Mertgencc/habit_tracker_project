### 🧠 Habit Tracker App

A simple and modern Habit Tracker application built with HTML, CSS and JavaScript.  
HTML, CSS ve JavaScript ile geliştirilmiş basit ve modern bir alışkanlık takip uygulamasıdır.

This project allows users to create habits, track them, filter by categories and store data in LocalStorage.  
Bu proje kullanıcıların alışkanlık eklemesine, takip etmesine, filtrelemesine ve verileri LocalStorage’da saklamasına olanak sağlar.

#### 🚀 Features / Özellikler

- ➕ Add new habits / Yeni alışkanlık ekleme  
- 🗂️ Category filtering system / Kategori filtreleme sistemi  
- ✅ Mark habits as completed / Alışkanlıkları tamamlandı işaretleme  
- 🗑️ Delete habits / Alışkanlık silme  
- 💾 Save data with LocalStorage / LocalStorage ile veri kaydetme  
- 📊 Progress tracking / İlerleme takibi

#### ⚙️ How It Works / Nasıl Çalışır

The application loads saved data from LocalStorage when opened.  
Uygulama açıldığında veriler LocalStorage’dan yüklenir.

Users can add, delete and complete habits dynamically.  
Kullanıcılar alışkanlık ekleyebilir, silebilir ve tamamlandı olarak işaretleyebilir.

Filtering system shows habits based on selected category.  
Filtreleme sistemi seçilen kategoriye göre verileri gösterir.

All updates are automatically saved in LocalStorage.  
Tüm değişiklikler otomatik olarak LocalStorage’a kaydedilir.

---

#### 📌 Purpose / Amaç

This project was built to strengthen JavaScript fundamentals such as DOM manipulation, events, filtering logic and LocalStorage usage.  
Bu proje DOM manipülasyonu, event kullanımı, filtreleme mantığı ve LocalStorage konularını pekiştirmek için yapılmıştır.
